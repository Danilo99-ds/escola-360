No final de cada período letivo, a coordenação pedagógica enfrenta um grande problema: consolidar relatórios, notas e observações de dezenas de professores em diferentes formatos. Além da burocracia que esgota os professores, a falta de dados centralizados impede a identificação precoce de alunos com baixo engajamento acadêmico, atrasando intervenções críticas.

O **Escola 360** foi desenvolvido como um hub central de gestão escolar. A plataforma foi projetada com foco na Experiência do Usuário (UX) do professor: a inserção de dados foi simplificada para garantir a adesão. Para a equipe pedagógica, o sistema oferece uma interface de busca unificada e painéis consolidados de cada estudante.

## Foco em Dados 
Este projeto é fundamentado na construção de um fluxo de dados eficiente para a gestão educacional:
* **Automação de Alertas:** Lógicas implementadas para monitorar a base de dados e gerar alertas automáticos sobre alunos com baixo engajamento, permitindo atuação preventiva.
* **Consolidação e ETL:** Processamento de dados descentralizados (inseridos pelos professores) em relatórios unificados e formatados para a tomada de decisão da gestão.
* **Monitoramento de Desempenho:** Estruturação dos dados para alimentar futuros dashboards, mapeando tendências de notas, faltas e comportamento por turma ou aluno individual.

## Tecnologias Utilizadas 
* **Front-end:** HTML5, CSS3, JavaScript.
* **Back-end & Banco de Dados:** Python, Django, .
* **Processamento de Dados:** Criação de scripts de automação para cruzamento de informações de turmas e disciplinas.
